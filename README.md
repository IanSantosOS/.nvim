# Neovim Config

Neovim Repository: <https://github.com/neovim/neovim>

## Setup

Windows:

```cmd
git clone https://github.com/IanSantosOS/.nvim.git $HOME\AppData\Local\nvim
```

Linux:

```sh
git clone https://github.com/IanSantosOS/.nvim.git $HOME/.config/nvim
```
